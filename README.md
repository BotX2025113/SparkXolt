# SparkXolt

**بات هوشمند SparkXolt** — ساخته شده توسط **بهارناز**

SparkXolt یک بات هوشمند و دوست وفادار است که:

- به سوالات هوش مصنوعی پاسخ می‌دهد
- اخبار روز را خلاصه می‌کند
- مثل یک دوست واقعی چت می‌کند
- مفاهیم پیچیده علم را ساده توضیح می‌دهد
- پیشنهاد کتاب، فیلم و انیمه می‌دهد

در حال توسعه توسط یک دختر ۱۲ ساله تیزهوش ایرانی!
